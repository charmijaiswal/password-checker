This is a simple Python program that checks password strength based on length and characters.

## Features
- Checks password length
- Detects uppercase and lowercase letters
- Checks numbers and special characters
- Gives strength output (Weak / Medium / Strong)

## How to Run
1. Make sure Python is installed
2. Run the file:
   python password_checker.py

## Example
Enter password: Hello123  
Output: Medium Strength
